## 📌 프로젝트 소개

냉장고 속 재료로 무엇을 만들지 고민하는 요리 초보자를 위한 레시피 검색 앱입니다.

- **자신만의 레시피 등록** — 나만의 레시피를 작성하고 다른 유저와 공유
- **단계별 조리 안내** — 초보자도 쉽게 따라할 수 있도록 순서대로 안내
- **식재료 기반 검색** — 보유한 재료를 입력하면 만들 수 있는 요리를 추천

---

## ✨ 주요 기능

### 🔍 레시피 추천 기능
- 다른 유저가 작성한 레시피 추천
- 재료 검색으로 맞춤 레시피 탐색
- 레시피 수첩에 찜한 레시피 저장

### 📝 레시피 등록 기능
- 직접 만든 레시피를 등록 및 저장
- 등록한 레시피를 다른 유저와 공유

## 🗂 시스템 구성도

```
시작 화면
└── 메인 페이지
    ├── 오늘의 추천메뉴 (랜덤 추천)
    ├── 재료 검색창
    ├── 레시피 검색
    │   ├── 최근 검색어
    │   └── 최근 본 레시피
    └── 레시피 수첩
        ├── 나의 레시피
        │   ├── 레시피 등록
        │   └── 레시피 상세보기
        ├── 찜한 레시피
        └── 프로필 편집
```

---

## 📱 구현 화면

| 화면 | 설명 |
|------|------|
| 메인 화면 | 오늘의 추천 메뉴(랜덤), 재료 검색창, 하단 네비게이션 |
| 레시피 검색 | 재료를 `/`로 구분하여 입력, 최근 검색어 표시 |
| 검색 결과 | 입력 재료 기반 요리 목록 안내 |
| 레시피 상세 | 1인분 기준 재료 안내 + 단계별 조리법 |
| 레시피 수첩 | 나의 레시피 목록 / 찜한 레시피 목록 |
| 레시피 등록 | 제목·설명·조리시간·재료·레시피 입력 후 저장 |
